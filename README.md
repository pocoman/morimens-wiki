## 아이콘 이미지 파일명 규칙

이미지는 각 항목의 영문명(`nameEn`)을 아래 규칙으로 변환한 파일명을 사용합니다.

**변환 규칙**: 소문자 변환 → `: ` `'` 등 특수문자 제거 → 공백을 `-`로 치환

| 영문명 | 파일명 |
|---|---|
| `Ramona: Timeworn` | `ramona-timeworn.png` |
| `Knight's Vow` | `knights-vow.png` |
| `Voices in Your Head` | `voices-in-your-head.png` |
| `Murphy: Fauxborn` | `murphy-fauxborn.png` |

### 저장 위치

```
assets/
├── characters/   ← 각성체 아이콘 (예: ramona-timeworn.png)
├── wheels/       ← 명륜 아이콘   (예: knights-vow.png)
└── keyflares/    ← 은열쇠 아이콘 (예: voices-in-your-head.png)
```

- **포맷**: PNG 또는 WebP
- **권장 크기**: 256×256px
- 이미지가 없는 항목은 빈 회색 박스로 대체 표시됩니다.
